# Cloud Strategy Template

## Overview

Reusable template for cloud strategy engagements. Covers assessment, planning, and roadmap development.

**Type**: Template (not a client project)
**Duration**: Typically 4-6 weeks
**Value**: 5,000 - 0,000

## Scope

### Phase 1: Current State Assessment (Week 1-2)
- Infrastructure inventory
- Application portfolio analysis
- Cost baseline
- Skills assessment
- Compliance requirements

### Phase 2: Target Architecture (Week 2-3)
- Cloud Platform selection (AWS, Azure, GCP)
- Architecture patterns
- Security model
- Network design
- DR/BC strategy

### Phase 3: Migration Roadmap (Week 3-4)
- Application prioritization
- Migration strategy (6 Rs)
- Timeline development
- Resource planning
- Risk assessment

### Phase 4: Business Case (Week 4)
- TCO analysis
- ROI projection
- Risk mitigation plan
- Executive Summary

## Deliverables

1. Current State Assessment Report
2. Target Architecture Document
3. Migration Roadmap (Gantt + priorities)
4. Business Case Presentation
5. Executive Summary

## Key Considerations

### Technical
- Hybrid vs full cloud
- Data residency requirements
- Integration complexity
- Skill gaps

### Business
- Budget constraints
- Timeline drivers
- Stakeholder alignment
- Change management
