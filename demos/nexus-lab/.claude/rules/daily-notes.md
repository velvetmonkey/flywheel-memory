# Daily Research Log Format

## Structure

Daily notes capture research progress with links to experiments, papers, and methods.

## Log Section

Format entries as continuous bullets:

```markdown
## Log

- 09:00 - [[Experiment-2026-01-15]] - Analyzed preliminary results
- 11:00 - [[PCR Protocol]] - Optimized annealing temperature
- 14:00 - [[Smith-2024-CRISPR]] - Reviewed methodology section
- 16:00 - [[Project Alpha]] - Updated timeline with new data
```

## Linking Conventions

- Link experiments: `[[Experiment-YYYY-MM-DD]]`
- Link methods: `[[Method Name]]`
- Link papers: `[[Author-Year-ShortTitle]]`
- Link projects: `[[Project Name]]`

## Research Notes

Include observations, hypotheses, and next steps at the end of each day's log.
