# Daily Notes Format

## Structure

Daily notes for startup operations include context for handoffs and automation triggers.

## Sections

### Log
Continuous bullets with timestamps and context:

```markdown
## Log

- 09:00 - [[Acme Corp]] - Onboarding call, walked through setup
- 11:00 - [[Customer Support Escalation]] - Resolved billing issue
- 14:00 - [[Weekly Sync]] - Product roadmap discussion
- 16:00 - [[TechStart Inc]] - Renewal conversation
```

### Wins
Celebrate daily wins:

```markdown
## Wins

- Closed [[BigCo]] annual contract ($24k ARR)
- Reduced support ticket backlog by 40%
```

### Handoff Notes
Context for automations or co-founders:

```markdown
## Handoff Notes

- [[Acme Corp]] needs follow-up email with API docs
- [[TechStart Inc]] wants demo of new feature next week
```

## Linking Conventions

- Link customers: `[[Customer Name]]`
- Link playbooks: `[[Playbook Name]]`
- Link meetings: `[[Meeting Name]]`
