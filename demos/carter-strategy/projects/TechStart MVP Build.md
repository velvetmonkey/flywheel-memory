## [[Key Decisions]]

1. **Framework**: [[Node.js]]/Express (team familiarity)
2. **Database**: PostgreSQL (relational needs, future scale)
3. **Hosting**: [[AWS ECS]] (cost-effective, scalable)
4. **Auth**: Custom JWT + OAuth (flexibility)

## [[Lessons Learned]]

- Scope creep in admin dashboard - should have fixed features earlier
- OAuth integration took longer than estimated
- Documentation handover critical for small teams
- Weekly demos kept stakeholder alignment
