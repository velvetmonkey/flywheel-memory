# [[Discovery Workshop Template]]

## Overview

Standard template for client discovery workshops. Used at the start of new engagements to understand requirements, constraints, and success criteria.

**[[Last Reviewed]]**: [[September 2025]]
**Duration**: 4-6 hours
**Participants**: Client stakeholders, project team

## Pre-Workshop

### [[Preparation Checklist]]
- [ ] [[Schedule 4]]-6 hour block
- [ ] Send pre-[[Read Materials]]
- [ ] Prepare whiteboard/collaboration tool
- [ ] Review existing documentation
- [ ] Identify [[Key Stakeholders]]

### Pre-Read Materials to Request
- Existing system documentation
- Organization chart
- Previous project artifacts
- Known pain points list

## [[Workshop Agenda]]

### 1. Introductions (30 min)
- Participant introductions
- Roles and expectations
- Workshop objectives

### 2. [[Current State]] (90 min)
- System walkthrough
- Data flow mapping
- Integration points
- Pain points discussion

*Break (15 min)*

### 3. [[Future State Vision]] (60 min)
- Business objectives
- Success criteria
- Non-negotiable requirements
- Nice-to-have features

### 4. Constraints & Risks (45 min)
- Timeline constraints
- Budget limitations
- Technical constraints
- Known risks

*Break (15 min)*

### 5. [[Stakeholder Mapping]] (30 min)
- Decision makers
- Influencers
- Users
- Communication preferences

### 6. [[Next Steps]] (30 min)
- Key takeaways
- Action items
- Timeline
- Follow-up scheduling

## Post-Workshop

### Deliverables (within 48 hours)
- [ ] Summary document
- [ ] Key findings
- [ ] Risk register (initial)
- [ ] Stakeholder map
- [ ] Proposed timeline

### Follow-up Meeting
- Review findings
- Validate understanding
- Refine scope
- Agree on approach

## Tips

- Start with open questions
- Let client talk 70% of the time
- Document everything visually
- Confirm understanding before moving on
- Identify decision makers early
