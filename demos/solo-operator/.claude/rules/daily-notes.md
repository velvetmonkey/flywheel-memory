# Daily Notes Format

## Frontmatter

Include energy and productivity tracking:

```yaml
---
energy: 7
focus: 8
revenue_today: 450
tasks_completed: 5
---
```

## Sections

### Briefing
Morning intentions and priorities

### Log
Continuous bullets tracking activities:

```markdown
## Log

- 08:00 - Newsletter draft for [[Content Calendar]]
- 10:00 - [[Client Call]] with potential sponsor
- 14:00 - Course recording: Module 3
- 16:00 - Ran [[Social Media Batch]] automation
```

### Wins
End of day wins and accomplishments

### Tomorrow
Quick notes for next day's priorities

## Linking Conventions

- Link content: `[[Content Calendar]]`, `[[Newsletter Draft]]`
- Link automations: `[[Automation Name]]`
- Link products: `[[Course Name]]`, `[[Product Name]]`
