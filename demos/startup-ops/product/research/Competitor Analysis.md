# [[Competitor Analysis

Competitive]] landscape for SMB analytics [[Tools]]

Owner: [[Jamie Patel]]
[[Last Updated]]: 2025-12-20

## [[Market Landscape]]

**[[Total Addressable Market]]:** 500K SMBs (50-500 employees) spending on analytics
**[[Served Market]]:** ~100K [[Using]] dedicated analytics tools ([[Not]] just spreadsheets)
**[[Our Target]]:** 10K SMBs (50-200 employees, $5M-$50M revenue)

## [[Direct Competitors]]

### 1. Tableau (by Salesforce)

**[[Target]]:** Enterprise + [[Mid]]-[[Market]]
**Pricing:** $70/user/mo (Viewer), $42/user/mo (Explorer), $70/user/mo (Creator)
**Strengths:**
- Market leader, trusted brand
- Powerful features, unlimited customization
- Strong enterprise integrations

**Weaknesses:**
- Complex setup (weeks to months)
- Requires data engineer or consultant
- Expensive for SMBs (min $700/mo for 10 users)
- Steep [[Learning]] curve

**Our advantage:** 10x [[Faster]] setup, 1/2 the [[Price]], no [[Data]] [[Team]] [[Required]]

### 2. Microsoft PowerBI

**Target:** Microsoft ecosystem companies
**Pricing:** $10/user/mo (Pro), $20/user/mo ([[Premium Per User]])
**Strengths:**
- Cheapest option
- [[Deep Microsoft]] [[Integration]] ([[Azure]], [[Office 365]])
- Familiar to Excel users

**Weaknesses:**
- Still requires setup/configuration
- Tied to Microsoft ecosystem
- [[Performance]] issues with large datasets
- [[Limited]] customer support

**Our advantage:** Better UX, faster setup, not locked to Microsoft

### 3. Looker (by Google)

**Target:** Mid-market to enterprise, [[Tech]] companies
**Pricing:** Quote-[[Based]] (estimated $3K-$10K/mo)
**Strengths:**
- Modern, code-based approach (LookML)
- [[Strong]] [[Data Modeling]]
- Embedded analytics [[Capabilities]]

**Weaknesses:**
- Very expensive
- Requires [[Technical]] team (LookML = code)
- Overkill for simple [[Use]] cases
- Slow implementation (months)

**Our advantage:** No-code, fraction of the cost, SMB-focused

### 4. Metabase ([[Open Source]])

**Target:** Startups, tech-savvy SMBs
**Pricing:** Free (self-hosted), $85/mo (hosted - 5 users)
**Strengths:**
- Free option
- Simple, clean UI
- Good for basic use cases
- Active open-source community

**Weaknesses:**
- Self-hosted = DevOps burden
- Limited features vs paid tools
- Basic support (community forums)
- Scaling issues

**Our advantage:** Fully managed, better features, proper support

## Adjacent Competitors

### 5. Google Data Studio (Looker Studio)

**Target:** SMBs, agencies
**Pricing:** Free
**Strengths:**
- Free
- Google ecosystem integration
- Easy sharing/embedding

**Weaknesses:**
- Limited data sources
- Basic features only
- Slow performance
- No SLAs or support

**Our advantage:** More powerful features, better performance, real support

### 6. Amplitude

**Target:** Product analytics (SaaS companies)
**Pricing:** $995/mo (Growth), [[Custom]] (Enterprise)
**Strengths:**
- [[Purpose]]-[[Built]] for product analytics
- User journey tracking
- Cohort analysis

**Weaknesses:**
- Narrow [[Use Case]] (product [[Only]], not [[General]] BI)
- Expensive
- Requires SDK integration

**Our advantage:** General-purpose analytics, [[Simpler]] setup

## Competitive Wins & Losses

### Recent Wins

**[[DataDriven Co]] (vs Tableau):**
- Why we won: "Tableau quote was $1,200/mo + $5K setup. You're $499 and we were live in [[One]] call."
- [[Key]] factor: Time to [[Value]]

**Prospect (vs PowerBI):**
- Why we won: "PowerBI requires our IT team. MetricFlow just works."
- Key factor: Self-serve, no dependencies

### [[Recent Losses]]

**[[InsightHub]] (to [[Specialized]] clickstream tool):**
- Why we lost: Performance on 10M+ rows
- Lesson: [[Don]]'t compete on [[High]]-volume data (yet)

**Enterprise prospect (to Tableau):**
- Why we lost: Required SSO/SAML (we don't [[Have]])
- Lesson: Enterprise [[Features]] matter for large deals

## Threats & [[Opportunities]]

**Threats:**
1. **PowerBI** could [[Improve]] UX and threaten our SMB position
2. **Metabase** could add [[Managed]] service and compete on price
3. **[[Big]] players** could [[Launch]] "[[Tableau Lite]]" for SMBs

**Opportunities:**
1. **Tableau fatigue** among SMBs - [[They]] want simpler
2. **Data democratization** [[Trend]] - more non-technical users [[Need]] analytics
3. **COVID [[Acceleration]]** - SMBs finally investing in data [[Infrastructure]]

## [[Differentiation Strategy]]

**[[Our Unique Value Prop]]:**
> "The fastest way for SMBs to [[Get]] from data to dashboards. No data team required, no months-long setup. Just connect your data and start [[Building]]."

**[[Key Messages]]:**
1. **Speed:** 30 minutes to [[First]] dashboard (vs weeks/months)
2. **Simplicity:** No SQL, no data team, no consultants
3. **Price:** Half the cost of alternatives ($499 vs $1K+)
4. **[[Focus]]:** Built for SMBs, not scaled-down enterprise tool
